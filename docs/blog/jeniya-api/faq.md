# jeniya.cn 常见问题

本文收集了 jeniya.cn API 使用中的常见问题。

## 账号相关

### 如何充值？

登录后进入"充值"页面，支持支付宝、微信支付，到账即时。

### 余额不足会怎样？

余额不足时 API 调用会失败。建议：
- 设置余额预警
- 定期查看用量统计
- 重度用户可考虑预充值

### 可以开发票吗？

可以。联系客服提供充值记录和开票信息即可。

### 账号可以共享吗？

不建议共享账号。每个用户应创建自己的令牌，便于：
- 安全管理
- 用量统计
- 问题排查

## 令牌相关

### 令牌泄露了怎么办？

立即执行以下操作：
1. 登录控制台
2. 找到泄露的令牌
3. 点击"删除"或"禁用"
4. 创建新令牌
5. 更新所有使用该令牌的地方

### 一个令牌可以用于多个项目吗？

可以，但建议：
- 为不同项目创建不同令牌
- 便于统计和管理
- 便于定位问题

### 令牌有有效期吗？

没有，令牌永久有效，除非：
- 手动删除
- 账号被停用

### 如何查看令牌信息？

令牌管理页面可以查看：
- 令牌名称
- 创建时间
- 最后使用时间
- 调用次数和用量

## 使用相关

### 为什么请求失败？

常见原因：

1. **令牌错误或已删除**
   - 检查令牌是否正确
   - 确认令牌未被删除

2. **余额不足**
   - 查看控制台余额
   - 及时充值

3. **请求格式错误**
   - 检查 API 文档
   - 对照示例代码

4. **网络问题**
   - 检查网络连接
   - 确认可以访问 jeniya.cn

建议：查看控制台的"调用记录"，可以获取详细的错误信息。

### 响应速度慢怎么办？

1. 检查网络连接
2. 尝试切换到稳定分组
3. 使用更快的模型（如 Flash 代替 Pro）
4. 减少输入文本长度
5. 联系客服反馈

### 支持哪些地区访问？

全球可访问，国内直连无需代理。

### 支持哪些模型？

支持 Claude、GPT、Gemini 及国产模型，详见 [支持模型](./models.md)

### 支持流式输出吗？

支持。所有主要模型都支持流式输出：
- Python：使用 `stream` 方法
- Node.js：使用 `stream` 方法
- REST API：设置 `stream: true`

### 有使用限制吗？

常规使用没有限制，但为防止滥用：
- 新用户有每日调用次数限制
- 高频使用需要升级账户
- 特殊情况可联系客服

## 成本相关

### 如何计算费用？

费用 = 输入 tokens × 输入单价 × 倍率 + 输出 tokens × 输出单价 × 倍率

例如：
- 输入 1000 tokens，单价 $0.03/1M，倍率 0.1x
- 输出 500 tokens，单价 $0.15/1M，倍率 0.1x
- 费用 = (1000/1M × $0.03 × 0.1) + (500/1M × $0.15 × 0.1) ≈ ¥0.001

### 如何降低成本？

1. **选择合适的模型**
   - 简单任务用 Flash/Haiku
   - 复杂任务才用 Pro/Opus

2. **优化提示词**
   - 减少不必要的输出
   - 明确指定输出格式
   - 使用系统提示词限制输出

3. **使用特惠分组**
   - 非关键任务用特惠分组

4. **充分利用缓存**
   - 相同请求避免重复调用

### 可以查看详细的费用明细吗？

可以。在控制台的"调用记录"中可以查看：
- 每次调用的详细费用
- 输入/输出 tokens 数量
- 使用的模型
- 时间戳

### 如何设置预算？

控制台支持：
- 设置每日/每月预算
- 余额预警设置
- 使用量限制设置

## 技术相关

### 支持哪些编程语言？

所有语言都支持，只要能发送 HTTP 请求：
- Python
- Node.js
- Java
- Go
- Ruby
- PHP
- ... 以及任何支持 HTTP 的语言

### 如何集成到现有项目？

1. 获取 API Key
2. 替换现有 API 地址
3. 修改请求头（如需要）
4. 测试调用
5. 更新生产环境

### 有 SDK 可用吗？

有。官方提供：
- Python: `anthropic`
- Node.js: `@anthropic-ai/sdk`
- Java: `com.anthropic`
- Go: `anthropic-ai/sdk`

### 有 API 文档吗？

有。详见：
- 在线文档：https://jeniya.cn/api-docs
- OpenAPI Spec：可在文档页面下载

## 集成相关

### 如何集成到 Claude Code？

1. 安装 Claude Code
2. 编辑 `~/.claude/settings.json`
3. 添加 API Key 和 Base URL 配置
4. 重启 Claude Code

### 如何集成到 Continue 插件？

1. 编辑 `~/.continue/config.json`
2. 添加 jeniya.cn 作为自定义模型
3. 配置 API Key 和地址
4. 重启 VSCode

### 如何集成到 Aider？

1. 创建 `~/.aider.conf.yml`
2. 配置模型和 API 信息
3. 运行 `aider`

### 如何集成到 Cursor？

1. 打开 Cursor 设置
2. Models → Add Model
3. 填写 jeniya.cn 配置信息
4. 保存设置
